# Product

## Register

brand

## Users

Founders early-stage, student entrepreneurs, y creadores multi-disciplinares entre 18-30 años que tienen múltiples identidades creativas (coder + designer, engineer + writer, entrepreneur + artist) y no encuentran herramienta para presentarlas de forma coherente. Contexto de uso: compartir su URL con potenciales inversores, clientes, colaboradores, o simplemente construir presencia pública. Job to be done: tener una presencia online que comunique quiénes son en su totalidad, no solo un rol.

## Product Purpose

Ingegno es una plataforma de perfiles públicos premium para "Da Vincis modernos". Permite crear una URL elegante que reúne proyectos, writing, skills y narrativa personal, con sincronización de GitHub y actualizaciones manuales. El modelo es freemium: hasta 2 proyectos gratis, Pro desbloquea ilimitados. El éxito se mide en: usuario crea perfil → lo publica → lo comparte activamente.

## Brand Personality

Renacimiento + tecnología. La palabra "ingegno" es el término que usaba Leonardo da Vinci para describir la inteligencia creativa que conecta disciplinas distintas. 3 palabras: *cinematic, intelligent, singular*. El tono emocional es confianza + distinción: el perfil tiene que sentirse mejor que LinkedIn, más auténtico que Linktree, más personal que GitHub. No es un CV ni un portfolio genérico — es una declaration of identity.

## Anti-references

- Linktree / Bento.me: demasiado informal, sin presencia
- Standard portfolio HTML: sin life, sin movimiento, sin narrativa
- LinkedIn: corporativo, rígido, impersonal
- Notion pages públicas: sin diseño intencional
- Cualquier SaaS dark mode con neon gradients y "AI purple"

## Design Principles

1. **Show, don't tell**: el propio producto tiene que demostrar calidad de diseño — si el perfil del usuario parece barato, el producto pierde credibilidad
2. **Cinematic by default**: dark mode first, animaciones de entrada con intención, tipografía grande y bold
3. **Restraint as luxury**: glassmorphism reservado para 5-6 superficies máximo, espacio en blanco generoso — el lujo se comunica con lo que NO está
4. **Renaissance texture**: la estética renacimiento (Da Vinci, tinta, cuadernos de bocetos) como contraste con la tecnología — no camp, no decorativo; auténtico
5. **One URL confidence**: el usuario tiene que sentir orgullo al compartir su enlace — el diseño tiene que sostenerse frente a cualquier audiencia

## Accessibility & Inclusion

WCAG AA mínimo. Dark mode first. Animaciones con prefers-reduced-motion alternatives. Focus-visible en todos los elementos interactivos. Contraste 4.5:1 para texto normal, 3:1 para texto grande.
